Selectores Combinadores

Selector descendiente

Para poder seleccionar un elemento especifico cuyo atributo pudiera ser compartido, vamos a usar el selector descendiente desde CSS para poder tomar esta ruta y aplicarla unicamente al elemento: 

                    
                        div .descendiente{
                            background-color: darkviolet;
                        }

Selectores hermanos

Este ejemplo es sencillo, a nivel de codigo, segun la jerarquia es como vamos a denominar como hermano o hijo al codiggo. Si encuentran al mismo nivel, son hermanos, si se encuentran por debajo o dentro, son hijos.

Para seleccionar hermanos en CSS vamos a seleccionar el primero y usaremos el signo "+" para decirle a nuestro navegador que debe seleccionar al hermano que está inmediatamente después del primero

Selector hijodirecto

Con este selector, vamos a poder darle estilo a los "hijos directos" de un div, los cuales serán los que están unicamente dentro de este msismo, en el caso de que el elemento estuivera dentro de otro div, ya no sería hijo, sino nieto y la regla ya no aplica. Para que esto funcione, debemos decirle al navegador que queremos solamente usar a los hijos de la etiqueta, esto se hace usando un ">" y después indicando que elemento de la contenedor queremos usar. 

                            
                                .container > p{
                                    background-color:
                                }
