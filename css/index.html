Practica 1

Vincular css a html

En este archivo vamos a empezar a trabjar las diferentes carcteristicas de css.
Para empezar, tenemos diferentes maneras de "encadenar" un archivo html a un css, la primera es que desde la parte de <head> agregamos la etiqueta <style> y dentro de ella vamos a ingresar el codigo.
La segunda manera es crear un documento tipo .css y crear un link a este desde el html que ya hemos creado.

Sintaxis CSS

La sintaxis se basa en los siguientes elementos

  • selector: El elemento al que le vamos aplicar los estilos
  • Propiedad: lo que vamos a cambiar
  • valor: el nuevo valor que voy a tener

Y graficamente queda así: 

                
                    selector{
                        propiedad:valor;
                        propiedad2:valor;
                        propiedadN:valor;
                    }
