Exsiten dos tipos de selectores

  • Selectores Simples
    1. Selectores Elementales
      Aquí vamos a encontrar el selector universal (*) y el selector de tipo o etiqueta (nombre de la etiqueta)
    2. Selectores de atributo id -> id del elemento:
      Vamos a reconocer un id mediante el atributo id dentro de una etiqueta, a su vez, en CSS lo vmos a seleccionar con un # para identificarlo, siempre usando de manera correcta el identificador, respetando las mayusculas y toda la gramatica del id, aunque no es recomendable usarlo ya que no es lo más adecuado a la hora de usar el codigo
    3. clase -> clase del elemento
      Para poder agregar estilo mediante las clases vamos a agregarle esta al elemento que necesitemos modificar, mediante el atributo class dentro de la etiqueta
      En CSS vamos a usar .class(nombre de la clase) para poder modificar el elemento
    4. Otros atributos: Para otros atributos vamos a usar las llaves cuadradas "[]" viendose así: 
                                  
                                [atributo]
                                [atributo=valor]
                                [atributo^=valor]
                                [atributo*=valor]
                                [atributo$=valor]

      En el primero sencillamete vamos a ingresar nuestro atributo "global", para que todos los elementos con este atributo sean modificados al mismo tiempo.
