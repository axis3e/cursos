Box Model

Width y Height

Empecemos trabajando con el width y el height en un modelo de caja, estos atributos nos ayudan a cambiar tanto el ancho como el alto de nuestra caja, recordemos que cada elemento de nuestro codigo se representa mediante cajas, independientemente del contenido que podamos encontrar dentro de la caja.

Elemento de bloque

Tambien recordemos que en html tenemos elementos de bloque (que son aquellos que usan toda la linea del navegador), a los cuales sí vamos a poder cambirles el tamaño y las especificaciones. 

                        
                            codigo html

                                <h2>Elemento de bloque</h2>
                                
                                <p>
                                    Ejemplo
                                </p>
                                

                            Código Css   
                            .block{

                            }

Elemento de linea

Mientras que al otro tipo de elemento, el sual se denomina "Elemento de Linea" no podemos cambiar su configuracion de ancho y alto, ya que abarca solamente el contenido que está dedntro de la etiqueta

Margin

Es la propiedad que nos permite generar espacio entre elementos
Es un shorthand (propiedad abreviada)que controla los 4 lados posibles a los que dar márgenes:

  • margin-top: Margen superior
  • margin-right: Margen derecho
  • margin-bottom: Margen inferior
  • margin-left: Margen izquierdo

Admite hasta 4 valores que van en el orden de las agujas de reloj, es decir: arrriba, derecha, abajo e izquierda.
Lo que quiere decir que podemos configurar de diferentes manera el margen, dependiendo del número de valores que agreguemos, entre más usemos, más especifica podrá ser eesta configuración:

  • 4 valores -> margin: top right bottom left
  • 3 valores -> margin: top right/left bottom
  • 2 valores -> margin: top/bottom right/left
  • 1 valor -> margin: top/right/bottom/left

A la hora de usar los margenes debemos tener cuidado y tención, ya que no funcionan igual en elementos de bloque y elementos de linea.
En este ejemplo (class:"bloque) podemos notar que al ser un elemento de bloque, este parrafo tiene margen en las 4 direcciones: 

                    
                        .bloque{
                            background-color: rgb(106, 80, 255);
                            margin: 100px 100px 100px 100px;
                        }

Para los elementos de linea, el valor "top" y el valor "bottom" no funcionan, ya que como se había mencionado antes, son elementos que etán sobre la misma linea y no pueden afectar a todo el bloque en el que se encuentran, así que al darles estilo de margin, quedan así:
Elemento .linea1Elemento .linea 

                    
                        .linea1{
                            color: coral;
                            background-color: rgb(84, 77, 185);
                            margin: 20px;
                        }
                        .linea{
                            color: rgb(31, 24, 83);
                            background-color: rgb(255, 255, 255);
                            margin: 10px;
                        }

Más elementos de margin

Algunas caracteristicas adicionales que podemos usar con margin es el poder darle un margen automatico a los elementos de bloque

Para esto, debemos haberle dado con anterioridad dimension al elemento de bloque, ya que si no lo hemos hecho, no será posible lograr lo deseado, ya que el elemento de bloque por defecto, abarca toda la página del navegador:

                        
                            .block3{
                                background-color: chocolate;
                                width: 300px;
                                height: 150px;
                                margin-left: auto;
                            }

Si usamos más de un solo margen por defecto (hablando de margenes horizontales) vamos a poder centrar el elemento de bloque

                        
                            .block4{
                                background-color: darksalmon;
                                color: #000;
                                width: 300px;
                                height: 150px;
                                margin-left: auto;
                                margin-right: auto;
                            }

Border

Es la propiedad que nos permite modificar el borde de la caja Es un shorthand (propiedad abreviada) que agrupa 3 propiedades

  • border-widhth: ancho el borde
    • border-top-width
    • border-right-width
    • border-bottom-width
    • border-left-width
  • border-style: estilo del borde
    • border-top-style
    • border-right-style
    • border-bottom-style
    • border-left-style
  • border-color: color del borde
    • border-top-color
    • border-right-color
    • border-bottom-color
    • border-left-color

Valores para estyle:

  • none (quita bordes)
  • hidden (quita bordes de manera vsual)
  • dotted (bordes de puntos)
  • dashed (borde de rectangulos)
  • solid (borde solido, comun)
  • double (borde doble)
  • groove (borde estilizado)
  • ridge (borde estilizado)
  • inset (borde estilizado)
  • outset (borde estilizado)

Box Sizing

Es la propiedad que nos permite controlar el cálculo que hace el navegador a la hora de modificar las propiedades content, padding y border. Los 2 valores que podemos darle son:

  • content-box -> Valor por defecto
  • border-box -> Cálculo de tamaño del elemento incluyendo el panding y el border

La función de esta etiqueta es la de determinar el tamaño máximo de nuestra caja, al usar el atributo border-box, vamos a tomar como refrencia el valor asignado a los bordes y de esta manera el navegador modifica el tamaño de la caja.

divi 
                        
                            divi{
                                width: 10em;
                                height: 10em;
                                background-color: brown;
                                margin-left: auto;
                                margin-right: auto;
                                margin-bottom: 3em;
                                box-sizing: border-box;
                                padding: 4em;
                            }
                            
                            .btn{
                                color: white;
                                background-color: red;
                                padding: 2em;
                            
                            }

Con esta etiqueta logramos que aún cuando el contenido cambia, el tamaño de la caja no lo hará.
