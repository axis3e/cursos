¿Qué es el DOM?

  • El DOM es el Document Object Model, lo que implica que es toda la estructura HTML del documento
  • No es parte de de JS, es más bien una API que conecta Js con HTML

La forma de representarlo es mediante un "arbol", del cual se desprenden diferentes ramas como en el ejemplo

Nodos

Cada parte del arbol, es un nodo.
Si bien podriamos llamarlo objeto (ya que cada etiqueta del html es un objeto), lo correcto es denominarle "nodo", y tenemos tres pricipales tipos de nodo:

  • Element Node (cualquier etiqueta HTML)
  • Text node (el contenido de la etiqueta)
  • Comment node (cualquier comentario en html)
