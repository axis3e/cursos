Especificidad

¿CÓMO FUNCIONA CSS?

Especificidad -> Establece cómo de específico es un selector para saber qué estilo aplicar.
El cálculo se realiza con la siguiente fórmula:

  • Etiquetas y pseudoelementos 001
  • Clases, atributos y pseudoclases 010
  • Ids 100
  • Estilos en linea 1000
  • !important GANA A TODO Y NO SE USA NUNCA

Para poner un ejemplo acerca de la especificidad vamos a usar el titulo de ejemplo, puesto que le hemos agregado tanto la clase como el id "title", y en base a ello, vamos a trabajar la especificidad a traves de CSS: 

                        
                            h1{
                                background-color: red;
                            }
                            /* specificity 0,0,1 */
    
                            .title{
                                background-color: green;
                            }
                            /* specificity 0,1,0 */
    
                            #title{
                                background-color: blue;
                            }
                            /* specificity 1,0,0 */
    
                            h1.title{
                                background-color: black;
                            }
                            /* specificity 0,1,1 */
                                                    
                            h1#title{
                                background-color: cornsilk;
                            }
                            /* specificity 1,0,1 */
                            
                            .title#title{
                                background-color: darkblue;
                            }
                            /* specificity 1,1,0 */
                            
                            h1.title#title{
                                background-color: blueviolet;
                            }
                            /* specificity 1,1,1 */

Podemos darnos cuenta mediante la especificidad que el estilo que tenga un valor más alto será el que se termine aplicando

Cascada

Cascada -> Funciona siempre que la especificidad sobre el elemento sea la misma
De la misma manera debemos aclarar que en caso de que las etiquetas llegasen a tener la misma especificidad van a ser cambiadas por la cascada, es decir, la que este más abajo va a sobreescribir a la que tenga encima.

Herencia

Herencia -> Es la capacidad que tienen algunos elementos de heredar algunas propiedades de sus elementos contenedores (padres, abuelos, etc)

En la mayoria de los casos, vamos a encontranos con caracteristicas heredadas, por ejemplo, al agregar un <span> dentro de un <p>, al agregarle un estilo a <p> el span va a heredar la caracteristica, ya sea tamaño, color, fuente, etc.

En el caso de que un elemento no heredase esta caracteristica y necesitasemos qeu así fuera tenemos el valor "inherit" para que así se logre:
