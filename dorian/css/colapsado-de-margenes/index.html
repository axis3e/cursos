Colapso de margenes

Existe un problema con los margenes, para el que actualmente no existe una solucion como tal, basada en documentación oficial. Este problema lo podemos ver reflejado en los margenes de los elementos de bloque:
Al querer dar una separación entre un elemento y el otro, usando margenes, estos no funcionan de la manera que se esperaria, si damos al primer elemento un margin-bottom de 20px y al segundo un margin-top de 10px se esperaria tener una separación de 30px, cosa que no sucede. Para solucionar este problema ded manera practica, se sugiere solamente usar la funcion de margin-bottom con el total de separación que se requiere

Otro de los problemas que vamos a encontrar se localiza en los titulos, los cuales tienen un margen extra por defecto, lo cual no permite que en caso de agrearle propiedades como por ejemplo de color, se marque por completo dicho color, dando un resultado diferente al esperado. La solución que se ha encontrado empiricamente es agregar un pequeño margen en css de quizas .1px para que esto se solucione
