Sintaxis de JavaScript

JavaScript es un lenguaje de programación que actualmente estaá presente en basicamente en todos los sitios web, ya que su función en ellos es el de darles interactividad.

En este curso vamos a aprender a usarlo y a trabajar con él incluyendolo en el html y el css.

Sintaxis

Para empezar debemos recordar que es case-sensitive lo que implica que distingue entre mayusculas y minusculas

El tipado es de tipo debil o dinamico, lo que implica que en una variable podemos incluir tanto numeros como letras

Una practica bastante recomendad es el de usar ";" al finalizar cada sentencia, ya que em otros lenguajes es necesario, aunque aquí no es obligatorio

Cada bloque se cierera con una llave de cierre "}" y de la misma manera opcionalmente podemos agregar el ";"

Una variable es un espacio que reservamos para almacenar datos que podemos cambiar durante la ejecucuión del programa, la manera correcta de declarase es con la keyword "let"

De igual manera, una cosntante es un espacio reservado pero para un dato que no va a cambiar en ningun momento, la keyword para declararla es "const"

El scope (ambito) es el espacio en el que nuestras declaraciones (let o const) existen, si en un bloque declaramos una let, ese será nuestro scope, fuera de este bloque, nuestra let no existe

Declaración, inicialización y modificación

Una variable se declara usando el siguiente comando

let varible;

Como podemos ver, abrimos declarando que tenemos una varible, seguido del nombre de nuestra variable y terminamos con un ";"


Para inicializar nuestra variable usamos la estructura

variable = valor;

Aquí se nota que la estructura es el nombre que anteriormente asiganamos a nuestra varable, seguido del simbolo "=" y con el valor de nuestra variable por delante


De la misma manera podemos declarar e inicializar nuestra variable en la misma sentencia:

let variable = valor;


Si quisieramos modificar una variable lo haremos de la siguiente manera:

variable = valornuevo;

Esto aplica siempre y cuando estemos dentro del scope de nuestra variable, fuera de ella marca error, ya que nuestra variable ya no existe.


Para el caso de las constantes solo podemos usar una sentencia para declarar e inicializarla, posteriormente no podemos hacer ninguna modificación, ya que son constantes los valores

const constante = valor;
